# Prenominal Adjective Order and Communication Context

## Metadata

- Authors: Joseph H. Danks, Mary Ann Schwenk
- Year: 1972
- Venue: Journal of Verbal Learning and Verbal Behavior, 11, 183–187
- DOI / URL: https://doi.org/10.1016/S0022-5371(72)80070-8
- Local PDF path: `10-writing/literatures/Danks and Schwenk - 1972 - Prenominal adjective order and communication context.pdf`
- Tags: adjective ordering, referential context, discriminability, pragmatics, forced-choice, English

## One-Paragraph Summary

Danks and Schwenk test whether the communication context — specifically, which adjective (color or size) discriminates a target referent from its competitors — influences the preferred ordering of prenominal adjectives. Using a forced-choice paradigm where participants selected between normal-order ("the large red car") and inverted-order ("the red large car") descriptions, they find that context-driven discriminability modulates ordering preferences, but only when phrases are presented with stress intonation. When color was the sole discriminating attribute, inverted (color-first) order was preferred; when size was the sole discriminating attribute, normal (size-first) order was preferred even more strongly than in control conditions. The results are interpreted as support for a pragmatic-communication rule: the more discriminating adjective tends to be placed first.

## Research Question

- Does the communication context (which adjective discriminates the target from distractors) influence preferences for adjective order, and is this effect modulated by intonation?

## Method / Data

- Design: Forced-choice experiment; 2 (Intonation: Stress vs. List) × 4 (Slide Type: Both Relevant, Neither Relevant, Color Relevant, Size Relevant) mixed factorial
- Participants/Data: 50 female undergraduates (native English speakers), Kent State University
- Measures: Proportion of inverted adjective order preferred per slide type
- Analysis approach: ANOVA + Newman-Keuls post-hoc comparisons + sign tests

## Key Findings

1. In the Stress intonation condition, participants strongly preferred color-first (inverted) order when color was the sole discriminating attribute (57% inversions), and preferred size-first (normal) order even more than control when size was the sole discriminating attribute (only 4% inversions — nearly at floor).
2. In the List intonation condition, none of the differences between slide types were significant; intonation interacted crucially with context.
3. The results support a pragmatic-communication rule: the more communicatively discriminating adjective is moved to first position; this overrides the default syntactic/semantic ordering when context demands it.

## Core Takeaways For The Current Project

- This is an early empirical demonstration that adjective ordering is not purely syntactically determined but is sensitive to referential context — a key premise of our RSA model.
- The discriminatory efficiency effect (more discriminating adjective goes first) is directly related to the DISCRIMINATORY STRENGTH hypothesis that Schlotterbeck and Wang (2023) pit against subjectivity. Our model instantiates discriminability through the RSA listener's referent identification process.
- The finding that the effect requires stress intonation suggests that non-canonical orderings carry prosodic marking, which is a boundary condition not modeled in our RSA framework.

## Limitations / Boundary Conditions

- Only size and color adjectives are tested; generalizability to other adjective classes (e.g., form/shape) is unknown.
- The forced-choice paradigm (choosing between spoken descriptions) is an indirect proxy for production; the pilot free-production task showed participants often omitted non-discriminating adjectives entirely.
- The experiment confounds word order and stress; no condition presented the inverted order without stress, so it is unclear whether the preference is for the order or for the prosodic marking.

## Useful Quotes (Short)

- "If the context necessitated a discrimination based on color, then Ss preferred the inverted order of adjectives. If size was the critical attribute, Ss strongly preferred the normal order." (Abstract)
- "The more discriminating adjective is moved to the primary position in order to enhance the probability of the listener making the correct identification." (p. 184)
