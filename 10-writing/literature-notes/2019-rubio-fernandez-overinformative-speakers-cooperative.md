# Overinformative Speakers Are Cooperative: Revisiting the Gricean Maxim of Quantity

## Metadata

- Authors: Paula Rubio-Fernandez
- Year: 2019
- Venue: Cognitive Science, 43(11), e12797
- DOI / URL: 10.1111/cogs.12797
- Local PDF path: `10-writing/literatures/Rubio-Fernandez - 2019 - Overinformative Speakers Are Cooperative Revisiting the Gricean Maxim of Quantity.pdf`
- Tags: overspecification, redundancy, cooperativeness, efficiency, audience design, Gricean maxims, referential communication

## One-Paragraph Summary

This paper develops an efficiency-based pragmatic account of referential communication as an alternative to standard Gricean informativeness accounts. Four production experiments show that speakers use redundant adjectives cooperatively: they do so more when color modification facilitates the listener's visual search (polychrome displays, Exp. 1a), but not when the listener already knows the target (Exp. 1b). English speakers use more redundant color adjectives than Spanish speakers due to the differential efficiency of prenominal vs. postnominal modification (Exp. 2), and the same pattern holds for redundant size adjectives (Exp. 3). The paper argues that redundant speakers are not egocentric but rationally sensitive to the listener's visual search needs, reframing cooperativeness as sensitivity to *discriminability* rather than *informativity*.

## Research Question

- Are speakers who use redundant adjectives egocentric or cooperative? Can overspecification be explained by efficiency (facilitating visual search) rather than violations of the Maxim of Quantity?

## Method / Data

- Design: Four language production experiments; referential communication task; object requests
- Experiment 1a/1b: Polychrome/monochrome displays; listener knows vs. does not know target
- Experiment 2: English vs. Spanish speakers; prenominal vs. postnominal adjective position
- Experiment 3: Redundant size adjectives
- Measures: Rate of redundant color/size adjective production
- Analysis approach: Mixed-effects logistic regression

## Key Findings

1. Speakers use redundant color adjectives more when they could facilitate the listener's visual search (polychrome displays) but not when the listener already has the target identified — showing listener-sensitivity, not egocentrism.
2. English speakers produce more redundant adjectives than Spanish speakers, confirming that prenominal position is more efficient for visual search.
3. The cooperativeness effect generalizes to redundant size adjectives.

## Core Takeaways For The Current Project

- Key theoretical move: replaces informativeness (number of competitors of same kind) with **efficiency** (ease of listener's visual search across entire display) as the operative criterion for referential choice.
- Distinguishes *egocentric* processing from *efficient* exploitation of visual common ground — directly relevant to how we frame the speaker model's rationality in RSA.
- The listener-knowledge manipulation (Exp. 1b) shows that efficiency is audience-designed, not automatic: RSA's speaker rationality assumption is empirically supported here.

## Limitations / Boundary Conditions

- Still production-only; efficiency hypothesis requires independent RT evidence (addressed in 2021).
- Cross-linguistic comparison (English vs. Spanish) limited to two languages.
- No formal computational model; efficiency is operationalized qualitatively through display design.
